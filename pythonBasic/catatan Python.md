## pytest CLI

Bakalan nge run file python yang nama file nya ada:
- test_xxx.py atau
- xxx_test.py
  
Run python test CLI nger run all file yang ada test nya
~~~
pytest 
~~~
kalo nge run file tertentu
~~~
pytest -q test_xxx.py
~~~
***OR***
~~~
pytest test_assert2.py
~~~
